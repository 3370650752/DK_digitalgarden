# web02:F12无法查看源代码
- 使用ctrl U
- 在地址栏前输入 `view-source:`，例如 `view-source:https://example.com`，然后按回车
- 使用 `curl` 或 `wget` 下载网页源代码，例如 `curl https://example.com`
- 在网页空白处右键点击，选择“查看页面源代码”或“查看源代码”（不同浏览器可能表述不同）
